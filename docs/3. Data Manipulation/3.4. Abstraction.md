# 3.4. Abstraction

# Functions
```python
def calculate_premium(base, risk, claim_history):
    premium = base * risk
    if claim_history:
        premium *= 1.2  # 20% surcharge for prior claims
    return premium

# Example usage
customer_premium = calculate_premium(500, 1.15, False)
print(f"Calculated premium: ${customer_premium}")
```
